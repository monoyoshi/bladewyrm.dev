hotaru(n)

Main page GitHub repository

A Discord bot that does a lot, from unserious things like laughing at people, to serious things like m*th. hotaru(n) is also evidently the predecessor to some of the other projects I have here. In a sense, he's like my sandbox.

Made using Javascript (Node.js) on a boring after-school November afternoon in 2016. He uses Discord.js to interact with Discord's API. And no, you can't add him onto your server yet (he's only hosted locally).

Oh, and the pages here double as a cooler, flashier, and more concise version of the help command.
